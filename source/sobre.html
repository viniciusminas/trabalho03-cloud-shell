Sobre o projeto

Infraestrutura cloud para um pequeno e-commerce.

Minas Store é um cenário fictício usado para demonstrar Docker, Ubuntu Server, Apache e automações com Shell Script em uma entrega simples, organizada e visualmente agradável.

Descrição do cenário

Do servidor ao deploy, tudo pensado como uma rotina operacional.

O projeto simula um ambiente de infraestrutura em cloud para uma loja virtual pequena. A aplicação é estática, mas o processo ao redor dela cobre tarefas comuns de administração Linux e DevOps, como instalação de serviços, organização de arquivos e validação do ambiente.

Voltar para a loja
Stack utilizada
  • Docker para containerização do ambiente.
  • Ubuntu Server como base do servidor Linux.
  • Apache para publicação do site estático.
  • Shell Script para automação das operações.
Recursos implementados

Rotinas que deixam o ambiente mais completo.

Atualização e instalação Preparação do sistema e validação do Apache.
Backup automatizado Cópia de segurança para os arquivos do site.
Deploy do site Publicação dos arquivos estáticos na estrutura do servidor.
Monitoramento Acompanhamento de processos, serviços e estado operacional.
Usuários e permissões Organização de acessos para manter o ambiente controlado.
Relatório operacional Registro final das etapas executadas no trabalho.
